iNaturalist vision saliency

Runs entirely in your browser with ONNX Runtime Web (WASM) on the same dequantized ONNX as the Python tool — weights match the official v25.01.15 INatVision_Small_2_fact256_8bit.tflite release. Preprocessing: 299×299, RGB 0–255. Saliency uses a SmoothGrad-style Monte Carlo estimate from forward passes only (TensorFlow.js tf.grad on a TF.js export of this graph fails in every backend we tested with a conv2d depth mismatch during backprop, so this demo does not use TF.js).

Technical note

The Python package uses PyTorch autograd for an exact gradient of the predicted softmax probability. This page approximates that quantity with forward-only ORT runs (SmoothGrad family). For exact ∂p/∂pixels matching production tooling, use npm run vision-saliency or python -m inat_vision_saliency in tools/inat_vision_saliency/.

Default photo: bear image from PlaceBear (saved in-repo for this demo).
