Technical note

The Python package uses PyTorch autograd for an exact gradient of the predicted softmax probability. ONNX Runtime Web does not expose reverse-mode autograd for arbitrary graphs, and exporting a single “gradient ONNX” from the onnx2torch stack is not currently viable, so the default demo approximates ∂p/∂pixels with SmoothGrad. For the exact map in a browser session, use the optional ?gradApi=… local server above, or run npm run vision-saliency / python -m inat_vision_saliency.

Default photo: bear image from PlaceBear (saved in-repo for this demo).
