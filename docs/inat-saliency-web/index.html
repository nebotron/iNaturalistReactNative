iNaturalist vision saliency

Runs entirely in your browser with TensorFlow.js and automatic differentiation (tf.grad) on a graph exported from the same dequantized ONNX as the Python tool. Weights match the v25.01.15 INatVision_Small_2_fact256_8bit.tflite release. Preprocessing: 299×299, RGB 0–255 (same as the mobile model). The page uses the CPU TensorFlow.js backend so tf.grad stays numerically reliable (WebGL can mis-handle conv depth during backprop on some graphs).

Technical note

The SavedModel input layout from the ONNX converter is [1, 299, 3, 299] (a transpose of NHWC). The page transposes your image into that layout for inference and maps gradients back to NHWC for display. Regenerate tfjs_model/ with tools/inat_vision_saliency/scripts/export_tfjs_graph_model.sh after rebuilding inat_vision_dequant.onnx.

Default photo: bear image from PlaceBear (saved in-repo for this demo).
