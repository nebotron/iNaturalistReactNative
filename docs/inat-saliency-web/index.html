Model file and GitHub Pages

Build inat_vision_dequant.onnx with the Python package in tools/inat_vision_saliency/ (see INTEGRATION.md there), then either select it above or place it next to this page as inat_vision_dequant.onnx so the app can fetch it on first load.

This page uses a forward-only Monte Carlo estimate of the gradient of the predicted class probability with respect to pixels (SmoothGrad-style). It is computed in the browser and is suitable for interactive demos; the Python pipeline uses full autograd for an exact Jacobian-style map.
